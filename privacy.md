# みてタッチ プライバシーポリシー

最終更新日: 2026年5月19日

「みてタッチ」（以下「本アプリ」）はユーザーのプライバシーを尊重します。本ポリシーは本アプリが扱う情報の内容と用途を説明します。

## 1. 取得する情報

### 1.1 カメラ・顔データ（TrueDepth API による視線追跡）
本アプリは視線で選択肢を選ぶアクセシビリティ機能を提供します。この機能のため、Apple の ARKit フレームワーク（`ARFaceTrackingConfiguration`、TrueDepth カメラを使用）から以下の **顔データ（face data）** を取得します。

**取得する顔データの内容:**
- 顔の位置と向き（`ARFaceAnchor.transform`）
- 視線が向いている推定方向（`ARFaceAnchor.lookAtPoint`）
- 瞬きの度合い（`ARFaceAnchor.blendShapes` の `eyeBlinkLeft`、`eyeBlinkRight`）

**取得しない情報:**
- カメラから取得される静止画像・動画は **一切取得しません**。
- 顔の写真、顔形状の3Dメッシュ、特徴点座標は **保存しません**。
- 個人を識別する情報（Face ID等の生体認証データ）には一切アクセスしません。

**利用目的:**
- 視線方向の判定（画面の左右どちらを見ているかを推定）
- 意図的な瞬き（0.3〜2秒の閉眼）を選択確定操作として検出
- 顔とデバイスの距離（25〜75cm 推奨範囲）を推定してユーザーに案内

**保存と保持期間:**
- 取得した顔データは **端末のメモリ上でフレーム単位で処理され、即時破棄** されます。
- ストレージ（端末内・iCloud・サーバー等）には **保存しません**。
- 保持期間は **ゼロ**(リアルタイム処理のみ)です。

**第三者への提供・外部送信:**
- 第三者への提供、外部送信は **一切行いません**。
- 本アプリはネットワーク通信機能を持たないため、技術的に外部送信は不可能です。

**ユーザーによる制御:**
- 視線追跡機能は設定でオフにできます。オフ時はカメラ・TrueDepth APIを一切使用しません。

### 1.2 写真
ユーザーが選択肢として登録した写真は、デバイス内のアプリ専用ストレージに保存されます。
- 外部送信・第三者提供は一切ありません。
- アプリを削除すると保存されたすべての写真も削除されます。

### 1.3 設定情報
注視時間、視線の感度、キャリブレーションプロファイル名などの設定は、デバイス内に保存されます。

## 2. 取得しない情報
本アプリは以下の情報を取得・収集しません。
- 氏名、メールアドレス、電話番号などの個人情報
- 位置情報、連絡先、カレンダー
- 広告識別子（IDFA）
- アプリ利用状況の分析データ（アナリティクス）
- ネットワーク経由でのデータ送信

## 3. 第三者への提供
本アプリは取得した情報を第三者に提供することはありません。インターネット通信機能を一切持ちません。

## 4. 子どもの利用について
本アプリは年齢制限 4+ で提供されています。特別支援教育を想定しており、保護者・支援者の方が安心してお使いいただける設計です。

## 5. ポリシーの変更
本ポリシーは予告なく変更される場合があります。重要な変更がある場合はこのページでお知らせします。
