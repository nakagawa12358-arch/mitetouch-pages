## よくある質問

### 視線が認識されません
- 設定 → みてタッチ → カメラが ON になっていることを確認してください
- 視線追跡は **TrueDepth カメラ搭載端末**（iPhone X 以降、または Face ID 対応 iPad Pro）で動作します
- 顔から画面まで **30〜60cm の距離** が最適です
- 強い逆光（窓を背にする等）は避けてください
- メガネの反射が強い場合は角度を変えてみてください

### キャリブレーションの仕方
1. 設定画面 → 「新規キャリブレーションを実行」
2. 画面の指示に従い、左→右の順に各3秒間視線を保つ
3. プロファイル名を入力して保存
4. 複数のお子様で利用する場合は、それぞれのプロファイルを保存可能

### 視線が使えないお子様への対応
- 設定 → 「視線で選択する」をオフ
- カメラを使わず、画面タップだけで操作できます
- タップモードでは何度でも選び直し可能です

### 写真を追加・編集したい
- 右上の📚アイコンでデッキ管理画面を開く
- 「+」で新規デッキ作成、または既存デッキをタップで編集
- １つのデッキに２枚以上の写真を追加できます

### 正解を設定したい
- デッキ編集画面の「正解（任意）」欄で正解の写真を選択
- 出題時は **正解の写真 + ランダムなディストラクタ** の組み合わせで表示
- 正解を選ぶとピンポン音が鳴ります

### 自動選択時間を変えたい
- 設定 → 自動選択タイマー で 5〜30秒の範囲で調整可能
- 視線が一方に向き続けると、設定した時間で自動的に選択されます
